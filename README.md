  - **Projects Hub**: Interactive cards showcasing real-world projects with tech stack badges and source links.
  - **Skills Library**: Radial or visual indicator graphs for core competencies.

  - **Education**: Timeline tracker for academic landmarks.
  - **Contact Portal**: Sleek interactive input form for messages.

## 🛠️ Tech Stack & Built With

### **Frontend & Interface**
*   **Framework:** [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) (for robust type safety)
*   **Build Tool:** [Vite](https://vite.dev/) (ultra-fast compilation and bundling)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) (utility-first responsive design)
*   **Animations:** [motion](https://motion.dev/) (fluid layout transitions, button hovers, page entry fades)
*   **Icons:** [Lucide React](https://lucide.dev/) (consistent, sleek vector icons)

### **Backend & AI Engine**
*   **Runtime:** Node.js (Express server backend)
*   **AI Integration:** `@google/genai` (SDK utilizing Google Gemini models server-side to power Mochi, keeping API credentials completely secure)
*   **Routing & Services:** Express with integrated Vite middleware for optimized client-server communication.

## 🚀 Getting Started (Run Locally)

### **Prerequisites**
Make sure you have Node.js (version 18+) installed.

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/kanishka12-tech/KanishkaVerse.git
   cd KanishkaVerse
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Gemini API Key (needed for Mochi AI Assistant):
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000` to preview the cosmic interface!

### **Production Build & Deploy**
To build and compile both the client assets and the server for production:
```bash
npm run build
npm start
```
